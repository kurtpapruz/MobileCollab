# Student Voting System

An Android application for managing student government and organization elections.

## Features

- Student Government (SG) Voting
  - Multiple positions with candidates
  - Edit votes before confirmation
  - Vote summary and confirmation
  
- Organization Officer Voting
  - Multiple positions with candidates
  - Edit votes before confirmation
  - Vote summary and confirmation

- Dashboard
  - View election results
  - Access voting sections
  - Profile management
